What is the {{$ctrl.AppSettings.getAppTitle()}}?

A portal for users of the {{$ctrl.AppSettings.getPlatformName()}}, currently providing self-serve tools for identity, onboarding and offboarding. With plans to provide more self-serve tools, docs, status and updates for the hosting platform, in one central place.

How do I create a user account on the hub?

Login (Menu > Login) using Keycloak and a new account will be created for you on the hub

What can I do on the hub?

Once you log in you can:

  • Manage your connected identities (currently: GitHub)
    • Needed to allow onboarding/offboarding to these services
  • View projects and team members

If you are made a project manager, you can:

  • Add someone to your project (on the project page)
  • Remove a team member from your project (using the Actions button next to a team member, on the project page)
  • Onboard and offboard team members on GitHub (using the Actions button next to a team member, on the project page). Note: you can only do this for users that have connected up their GitHub identity within the hub.
Why can't I create new projects or perform certain actions?
You need to be granted specific permissions to be able to do certain things on the hub
  • Project managers can manage a project's team and onboard/offboard team members
  • Admins manage the users and projects on the hub and grant special permissions to others
How do I request support?

If you can't achieve what you need via self-serve in the hub (see FAQs above) then you can request support by filling in one of the available support request forms.
